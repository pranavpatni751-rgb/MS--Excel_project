# 📊 Excel Sales Dashboard

An interactive Microsoft Excel dashboard designed to analyze and visualize sales data. This project leverages Excel's advanced features to transform raw data into meaningful insights, helping users monitor performance and support data-driven business decisions.

---

## 📌 Project Overview

This project demonstrates the use of Microsoft Excel for data cleaning, analysis, and dashboard creation. It provides an interactive view of sales performance using Pivot Tables, Pivot Charts, Slicers, and other Excel features.

---

## 🎯 Objectives

- Clean and organize raw sales data.
- Analyze sales performance across different dimensions.
- Create an interactive dashboard for easy data exploration.
- Identify trends and business insights.
- Support data-driven decision-making.

---

## 🛠️ Tools & Features Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning
- Sorting & Filtering
- Formulas & Functions (SUMIFS, COUNTIFS, IF, VLOOKUP/XLOOKUP)
